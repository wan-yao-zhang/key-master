# 🔑 Key Master - 密码生成器

一个安全、免费、开源的在线密码生成工具，帮助用户快速生成符合安全标准的强密码。

## ✨ 特性

- 🔒 **安全生成**：使用 Web Crypto API 生成加密安全的随机密码
- 🎨 **现代化界面**：简洁美观的 UI，支持暗色/亮色主题切换
- 📱 **响应式设计**：完美适配桌面端和移动端
- ⚡ **快速生成**：毫秒级响应，即点即用
- 🎯 **自定义选项**：
  - 密码长度：8-128 位（默认 16 位）
  - 字符类型：大写字母、小写字母、数字、特殊字符
  - 高级选项：排除相似字符、排除歧义字符
- 📊 **密码强度评估**：
  - 实时显示密码强度等级（弱/中/强/极强）
  - 显示密码熵值（Entropy）
  - 估算破解时间
  - 密码组成分析
- 📋 **批量生成**：支持一次生成多个密码（最多 100 个）
- 💾 **导出功能**：批量导出为 CSV 文件
- 🌐 **PWA 支持**：可安装为桌面应用，支持离线使用
- ⌨️ **键盘快捷键**：
  - `Ctrl/Cmd + Enter`：生成密码
  - `Ctrl/Cmd + C`：复制密码

## 🚀 快速开始

### 本地运行

1. 克隆或下载项目
2. 使用任意 HTTP 服务器打开 `index.html`

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

3. 在浏览器中访问 `http://localhost:8000`

### 部署

项目是纯静态网站，可以部署到任何静态网站托管服务：

- **GitHub Pages**：推送到 GitHub 仓库，在设置中启用 Pages
- **Vercel**：连接 GitHub 仓库，自动部署
- **Netlify**：拖拽 `index.html` 或连接 Git 仓库
- **其他**：任何支持静态文件的服务器

## 🎯 使用说明

### 生成密码

1. 调整密码长度（滑块或直接输入）
2. 选择需要的字符类型
3. 点击"生成密码"按钮
4. 点击"复制"按钮复制密码

### 批量生成

1. 点击"批量生成"按钮
2. 设置生成数量（1-100）
3. 点击"生成"按钮
4. 可以单独复制每个密码，或点击"导出为文件"批量导出

### 快速预设

- **强密码**：包含所有字符类型，16 位
- **中等密码**：包含字母和数字，12 位
- **PIN码**：仅数字，6 位
